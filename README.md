# Replication Package

**Arvidsson, Bello & Keuschnigg — *The Structural Origins of the Conservative Online Media Niche, US Twitter 2022***

This repository contains all data and code necessary to reproduce the figures in the main article and appendix. Raw data cannot be shared due to privacy constraints, as well as practical infeasibility (raw data are on the order of hundreds of gigabytes).
